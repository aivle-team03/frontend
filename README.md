# BOSS Frontend

다현님의 부재로 인한 산업안전 관리 AI 플랫폼 프론트엔드 임시 레포 

## 기술 스택

- React
- JavaScript
- CSS

---

## 실행 방법

의존성 설치:

```bash
npm install
```

개발 서버 실행:

```bash
npm run dev
```
