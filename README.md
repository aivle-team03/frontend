# BOSS Frontend

다현님의 부재로 인한 산업안전 관리 AI 플랫폼 프론트엔드 임시 레포 

## 주요 기능

- 관리자 대시보드
- CCTV 모니터링
- 위험 탐지 이벤트 확인
- 조치 요청 및 이력 관리
- 안전 법규 Q&A

## 기술 스택

- React
- JavaScript
- CSS

---

## 홈 화면 구조

현재 홈 화면은 BOSS 산업안전 관리 플랫폼의 초기 관리자 대시보드 골격만 간단히 구성한 상태입니다.

- 왼쪽 고정 사이드바
- BOSS 로고 영역
- 사이드바 탭 메뉴 및 접기 버튼
- 오른쪽 상단 헤더
- 향후 카드, 통계, CCTV 모니터링, 체크리스트 등을 추가할 수 있는 빈 콘텐츠 영역

## 실행 방법

의존성 설치:

```bash
npm install
```

개발 서버 실행:

```bash
npm run dev
```
